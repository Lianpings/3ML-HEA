This folder is for prediction of YM and CRSS for HEA and optimize alloys with high YM and CRSS:
paper published in Small: https://pubmed.ncbi.nlm.nih.gov/34524736/ (10.1002/smll.202102972)

Data folder: original data and cleaned data
Models folder: trained model

1data-clean.ipynb: clean data
2HEA-YM.ipynb: choose features for YM saved in Data folder, build YM model and train model saved in Models 
3HEA-CRSS.ipynb: choose features for CRSS saved in Data folder, build CRSS model and train model saved in Models
4YM-CRSS-HEA-opt.ipynb: optimize high YM and CRSS alloys using GA algorithm
